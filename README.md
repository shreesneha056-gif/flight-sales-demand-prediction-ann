> ANN-based regression model to forecast flight ticket sales demand from historical booking data.

---

## 📌 Problem Statement
Airlines need accurate demand forecasts to optimize pricing, seat allocation, and scheduling. This project builds a deep learning model to predict future flight sales volume.

## 🎯 Key Results
| Task | Technique |
|------|-----------|
| Regression | Sequential ANN (Keras) |
| Preprocessing | Feature scaling, train-test split |
| Evaluation | MAE, RMSE, R² Score |

## 🛠️ Tech Stack
- **Language:** Python
- **Deep Learning:** Keras, TensorFlow
- **Data Processing:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn
